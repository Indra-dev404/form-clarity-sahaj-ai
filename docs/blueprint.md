# **App Name**: Sahaj AI

## Core Features:

- Document Upload and Processing: Allow users to upload government forms (PDF, JPG, PNG). Perform client-side validation and temporary storage in Firebase Storage before sending to the AI for analysis.
- AI-Powered Explanation: Use the Google Gemini API as a tool to extract key information, simplify complex language, and provide clear explanations of government forms.
- Multilingual Support: Offer explanations in English, Hindi, and Bengali, Marathi, Gujrati and Tamil with automatic language detection.
- Voice Narration: Enable text-to-speech functionality for explanations with playback controls.
- User Experience Enhancements: Implement features like a clean, distraction-free interface, print-friendly explanation view, and session-based recent uploads.

## Style Guidelines:

- Primary color: Saturated cobalt blue (#2563EB), reflecting the government theme while avoiding cliche. Conversion from HSL(221, 74%, 53%)
- Background color: Very light grayish-blue (#F0F4FF). Conversion from HSL(221, 74%, 96%)
- Accent color: Moderate greenish-blue (#0EA5E9). Conversion from HSL(197, 81%, 45%)
- Headings: 'Space Grotesk' sans-serif, for a modern, official feel. If longer text is anticipated, use 'Space Grotesk' for headlines and 'Inter' for body
- Body text: 'Inter' sans-serif, for readability and a clean aesthetic.
- Use Lucide React icons for a consistent and modern look throughout the platform.
- Implement a responsive, mobile-first design with a clean, intuitive layout for easy navigation.
